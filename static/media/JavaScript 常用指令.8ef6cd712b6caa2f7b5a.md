## 基本指令

- code

  ```
  // declare
  let a = 0 // 區域變數
  var b = 1 // 全域變數 (盡量不使用)
  const c = 3 // 常數

  // operator
  1 == 1 // true
  1 == '1' // true
  1 === '1' // false
  ```
